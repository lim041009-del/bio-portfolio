Project

진행한 4개의 생명과학 프로젝트를 핵심 질문, 접근 방식, 기대 성과로 정리했습니다.

01

세포생물학

세포의 구조적 특징과 기능적 역할을 분석하며 세포주기, 세포막, 세포 소기관의 관계를 탐구했습니다.

  • 세포 구조 관찰
  • 세포주기 변화 정리
  • 소기관 기능 비교
02

분자생물학

유전 정보의 저장과 발현 과정을 중심으로 전사, 번역, 유전자 조절의 핵심 원리를 정리했습니다.

  • DNA-RNA 흐름 분석
  • 단백질 합성 과정 정리
  • 유전자 발현 조절 탐구
03

생화학

효소, 대사, 생체분자의 반응성을 연결해 생명 활동을 화학적 관점에서 이해했습니다.

  • 효소 반응 원리
  • 대사 경로 비교
  • 생체분자 특성 정리
04

신호전달

외부 자극이 세포 내부 반응으로 이어지는 과정을 수용체, 신호 단백질, 전사 반응 중심으로 분석했습니다.

  • 수용체 활성화
  • 신호 경로 모델링
  • 세포 반응 해석
